# computer-network-project
university computer network project
Hotel Network Project

Overview

This project demonstrates the design and implementation of a hotel network using Cisco Packet Tracer. The network is divided into two separate LANs: a Guest Network and a Staff Network, providing better security, organization, and network performance.

Network Structure
- Guest Network: 192.168.1.0/24
- Staff Network: 192.168.2.0/24
- Serial Link: 10.0.0.0/30

Each network includes:
- 1 Router
- 1 Switch
- Multiple PCs
- 1 DNS Server

Routing

Static routing was configured to enable communication between the Guest and Staff networks through a serial connection between the two routers.

Testing Results

- Successful communication within the same network.
- Successful communication between both networks.
- Successful DNS resolution.
- 0% packet loss during connectivity tests.

Purpose

The purpose of this project is to separate guest and staff operations in a hotel environment, improving network security, organization, and performance while allowing secure communication between the two networks.

Tools Used

- Cisco Packet Tracer
- Static Routing
- IPv4 Addressing
- DNS Configuration
